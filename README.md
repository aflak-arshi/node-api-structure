# node-api-structure
Node JS basic api structure

# Getting started

# Prequities
  1) [Node JS](https://karma-runner.github.io).
  2) [MongoDB](https://www.mongodb.com/)

## Installation
```sh
$ git clone https://github.com/aflak-arshi/node-api-structure.git
$ cd node-api-structure
$ npm i
```

## Load env variables
cp .env-example .env

## Development
```sh
$ npm start
```
